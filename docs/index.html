API開発者向けドキュメント

このページでは、システムの各機能やAPIの利用方法について説明します。モダンで使いやすいインターフェースを提供します。

GET /api/v1/users

登録されているユーザーのリストを取得します。ページネーションとフィルタリングをサポートしています。

POST /api/v1/export

データをエクスポートするリクエストを送信します。非同期処理としてバックグラウンドで実行されます。

GET /api/v1/status

現在のシステムのステータスと、稼働状況に関する詳細なヘルスチェック情報を返します。
